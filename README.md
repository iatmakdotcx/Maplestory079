## Maplestory079

### 1. 环境安装

开发工具 IntelliJ IDEA

JDK 11

### 2. 数据库
Mysql 或 MariaDB 

还原```sql/v079.zip```



### 3. 配置 settings.ini
server.settings.ip = 你的IP

server.settings.db.password = 你的数据库密码



### 3.运行

server.Start

### 4.客户端

 1. 将`tools/client/HShield.rar`解压到客户端替换`HShield`。
 2. 修改`MapleStory.bat`中的IP，并复制到客户端。
 3. 双击运行`MapleStory.bat`
